## Customizations
Things that can be customized:

### Images
- Life icon
- Ball texture
- Wall texture
- Collectible texture
- Floor texture

### Sounds
- Music
- Collect
- Hit

### Game Settings
- All game text 
- Starting lives
- Score gain
